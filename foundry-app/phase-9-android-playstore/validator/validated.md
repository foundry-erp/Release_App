## Deliverable 6: Generate mipmap icons

**Prerequisite:** Deliverables 1–5 and Pre-Flight Gate must be complete.

**Command to run (from `flutter/` directory):**
```bash
cd "C:/Users/bijay/OneDrive/Desktop/release_app/foundry-app/flutter"
dart run flutter_launcher_icons
```

**What this does:** Reads `flutter_icons:` config from pubspec.yaml → reads `assets/icon/icon.png` → generates `ic_launcher.png` at all mipmap densities:
- `mipmap-mdpi/ic_launcher.png` (48×48)
- `mipmap-hdpi/ic_launcher.png` (72×72)
- `mipmap-xhdpi/ic_launcher.png` (96×96)
- `mipmap-xxhdpi/ic_launcher.png` (144×144)
- `mipmap-xxxhdpi/ic_launcher.png` (192×192)

**Note:** These mipmap directories already exist (confirmed by reading filesystem). The command will overwrite the current default Flutter launcher icon with the Foundry icon.

**Pass condition for the command:** Exit code 0, no ERROR lines in output.

**If the command fails with "dart run" not found:** Try `flutter pub run flutter_launcher_icons` instead.

**Acceptance Criteria:**
- AC-9.6: `ls -lh flutter/assets/icon/icon.png` → file listed, size >= 50K
- AC-9.7: `ls flutter/android/app/src/main/res/mipmap-xxxhdpi/` → contains `ic_launcher.png`

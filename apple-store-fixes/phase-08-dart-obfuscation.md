# Phase 08 — Enable Dart Obfuscation for Release Builds

## Why This Exists
The current build config has `DART_OBFUSCATION=false`. While not a direct rejection reason, enabling obfuscation:
1. Makes reverse engineering harder (security best practice)
2. Reduces binary size slightly
3. Shows professionalism in release build configuration
4. Prevents potential issues with Apple's automated binary scanning

## Goal
Enable Dart obfuscation and tree shaking for release builds, and ensure debug symbols are preserved for crash reporting.

## Core Architecture Constraint
DO NOT modify any Dart source code, WebView logic, module loading, or local HTTP server. This phase only changes build configuration flags.

## Context — Current Build Config
**File:** `foundry-app/flutter/ios/Flutter/Generated.xcconfig`
- `DART_OBFUSCATION=false`
- `TREE_SHAKE_ICONS=false`
- `TRACK_WIDGET_CREATION=true`

**Note:** `Generated.xcconfig` is auto-generated by Flutter. Changes here get overwritten on `flutter build`. The correct way to enable obfuscation is via build flags in the build command or in a build configuration file.

## What To Build

### Deliverable 1: Create Build Script for iOS Release
**Location:** `foundry-app/flutter/scripts/build_ios_release.sh`

Create a shell script that builds the iOS release with proper flags:

```bash
#!/bin/bash
# Build iOS release with obfuscation and debug symbols
# Usage: ./scripts/build_ios_release.sh

set -e

echo "Building iOS release with obfuscation..."

flutter build ipa \
  --release \
  --obfuscate \
  --split-debug-info=build/debug-info/ios \
  --tree-shake-icons

echo "Build complete!"
echo "Debug symbols saved to: build/debug-info/ios/"
echo "IPA location: build/ios/ipa/"
echo ""
echo "IMPORTANT: Keep build/debug-info/ios/ for crash report symbolication."
echo "Upload these symbols to Firebase Crashlytics or similar service."
```

### Deliverable 2: Create Build Script for Android Release
**Location:** `foundry-app/flutter/scripts/build_android_release.sh`

```bash
#!/bin/bash
# Build Android release with obfuscation and debug symbols
# Usage: ./scripts/build_android_release.sh

set -e

echo "Building Android release with obfuscation..."

flutter build appbundle \
  --release \
  --obfuscate \
  --split-debug-info=build/debug-info/android \
  --tree-shake-icons

echo "Build complete!"
echo "Debug symbols saved to: build/debug-info/android/"
echo "AAB location: build/app/outputs/bundle/release/"
```
